# Página de entretenimiento

En este proyecto, intenté crear una página interactiva de Pokémon donde los usuarios pueden registrarse y, posteriormente, cuando mejore mi código, podrán comprar Pokémon.



## Tecnologías Utilizadas

- JS
- React
- React-Bootstrap
-MongoDB


## Funcionamiento

El sitio consta de 5 secciones, donde la barra de navegación y el footer se mantienen consistentes:

1. **Home:** En esta sección busqué simplemente hacer una introducción a lo que sería la página en general, trabajando con imágenes y sus descripciones, siempre tratando de que el diseño sea responsivo.

2. **Registro:**En esta sección, el usuario debe registrarse con un nombre, colocar su correo electrónico y una contraseña. Estos datos posteriormente irán a la base de datos donde se guardarán.

3. **Login:** En esta sección es donde se corrobora que dicho usuario esté registrado. En caso afirmativo, se redirige a la ventana de Pokémon; caso contrario, se redirige a la parte de registro.

4. **Pokemons:**En esta sección, se pueden ver los Pokémon iniciales con los que podría empezar un usuario y, en caso de querer informarse más, pueden acceder a los datos de los Pokémon apretando el botón de mostrar en el Pokémon respectivo y se envía a una ruta específica del mismo.

5. **Entrenadores:** En esta parte se ve una lista de todos los usuarios que se registraron, junto a su correo electrónico. La idea es que el usuario pueda ver esto, por lo que no puse más datos a la vista.

6. **Icono:**: El icono de la pokébola está para realizar funciones de administrador. En caso de hacer clic en la pokébola de la barra de navegación, se envía a la ruta donde se completa el CRUD completo de los usuarios registrados, ya que se pueden eliminar y actualizar.

## Observaciones

Se respetaron los colores y se priorizó el diseño responsivo. La elección de los colores fue para que se asemeje a una pokébola, por eso la mitad de rojo y la otra mitad de blanco, alternando con negro en ciertas partes para mejor visibilidad.
